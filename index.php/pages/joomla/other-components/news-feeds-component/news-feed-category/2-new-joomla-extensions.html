New Joomla! Extensions

Find extensions for your Joomla site in the Joomla Extensions Directory, the official directory for Joomla components, modules and plugins.

  1. Joocial Composer App

    Joocial Composer App is a brand new mobile App for Joomla! social management.

    This is a mobile application (not a Joomla extension), powered by open source technology: Apache Cordova/Phonegap, AngularJS and Ionic Framework.

    Joocial Composer App is a mobile app integrated with AutoTweet Joocial for social management. A Phonegap/Cordova application, based on Ionic (an advanced HTML5 hybrid mobile app framework), powered by AngularJS.

    • iOS and Android
    • Portrait
    • OAuth token based authorization
    • Local database, to persist information.

    In a nutshell, Joocial Composer App has the core features to create and publish posts from Joomla!

    • Control Panel and Stats: A control panel, to show general operation stats
    • Composer Editor: An advanced form to enter or edit social posts, with custom control types.
    • Requests list: A list of posts: to navigate and manage social posts

    Display Weather Forecasts of Current Time and Full Weeks of Any Zone You Want.

    Employing very stable and powerful Open Weather API, its a nice module that allows you to display weather Forecasts of current time as well as Forecasts of full weeks. You can display of any zone of the World. You just need to put Area/City name, It will display the forecast in details. Weather conditions are available in different amazing icons. You have the full control to display different parameters as well as to customize the styles.

  3. Glyphicons for Ark Editor

    This is a simple plugin, that allows you to add Bootstrap 3 Glyphicons to your content. Glyphicons are scalable graphics that can be easily customized for any colour and size.

    Please note that this plugin is dependent upon Bootstrap 3. Please check with your template provider for supporting compatibility.

  4. Smart Instagram Feed Widgets

    Build a Nice Picture Gallery with your Instagram Photos Using Several Widgets With Different Effects.

    A simple module to display images from Instagram to the Joomla websites as Picture Gallery. This module Fetches Instagram Feeds Using Several Instagram Widgets(SnapWidget, LightWidget & Uptsi). You can use any of them OR you can use all. To configure this module you will need Widget Keys. you can put custom code as well.

  5. TSI Animate Chart

    TSI Animate Chart is an extremely malleable module. The module allows in an effective way, introduce your data. The module is very easy to use, and extremely functional. The module also has a ready-made layouts.

    Main features of the module:
    - Templates diagrams;
    - Possibility to use custom colors;
    - Defining the font size in numbers, suffixes and prefixes;
    - Defining colors;
    - Bold font;
    - The width of the diagram;
    - The thickness of the track;
    - The spacing between lines;
    - Types of charts: circle, 3/4, a semi-circle;
